# Furnace
A small fps game project im working on in the Godot game engine. As a long term goal, I would like this game to be a fps and rts hybrid where players fight to destroy eachother's furnace in a cold arctic environment. 

Some of my long term goals for this project is having:
- Multiplayer functionality
- A fun and enganging movement system
- Economy and resource managment
- Npc's that follow basic tasks directed by player
